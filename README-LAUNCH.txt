EmailRank launch package

Files:
- index.html: homepage
- about.html: About page
- contact.html: Contact page
- privacy.html: Privacy Policy
- gmail-vs-proton.html: comparison page
- outlook-vs-gmail.html: comparison page
- best-email-for-privacy.html: editorial page
- fastmail-review.html: review page
- custom-domain-email.html: editorial page
- robots.txt
- sitemap.xml

Before launch:
1. Replace every instance of https://your-domain.com with your real domain.
2. Replace contact@your-domain.com with your real email.
3. Add your analytics script in index.html and other pages if desired.
4. Upload social-preview.png to the root if you want link previews.
5. Deploy the folder to Vercel or Netlify.
